# 📂 RAG 시스템 검증용 테스트 데이터셋 가이드

본 가이드는 프로젝트 빌드 후 pgvector 저장소 및 LLM의 컨텍스트 매핑 능력을 검증하기 위한 표준 문서 묶음과 저격용 테스트 질문 리스트입니다. 모든 질문은 제공된 가이드북의 실제 수치 및 예외 조항을 기반으로 설계되었습니다.

## 2. RAG 파이프라인 검증용 질문 리스트 (Scenario Test)

테스트 화면(`/chat`)에서 아래 질문들을 입력하여 시스템의 **[정밀 수치 매칭 / 조건부 필터링 / 다중 문서 조립 / 환각(Hallucination) 방지]** 성능을 검증합니다.

### 📌 [LEVEL 1] 정밀 수치 및 팩트 매칭 테스트
> AI가 외부 지식을 창작하지 않고, pgvector에서 가져온 문서 내의 정확한 통계치와 벌칙 수치를 인용하는지 검증합니다.

* **Q1. 서류 보존 기간 및 벌칙 규칙 (문서 3 저격)**
    - **질문:** `"근로자 명부와 근로계약에 관한 중요 서류는 사업장에 몇 년간 보존해야 하며, 이를 위반했을 때의 벌칙이나 과태료 수위는 어떻게 되나요?"`
    - **통과 기준:** **'3년간 보존'**, 위반 시 **'500만 원 이하의 과태료'**가 본문에 명시된 대로 오차 없이 출력되어야 함.
* **Q2. 해고 예고 예외 수당 (문서 3 저격)**
    - **질문:** `"근로자를 해고하려면 최소 며칠 전에 예고를 해야 하나요? 만약 이 기간을 준수하지 않고 즉시 해고할 때 회사가 근로자에게 지급해야 하는 수당의 정확한 법적 명칭과 지급 기준을 서술해 주세요."`
    - **통과 기준:** **'30일 전 예고'**, 미준수 시 **'30일분 이상의 통상임금(해고예고수당)'** 지급 필수 조항이 도출되어야 함.

### 📌 [LEVEL 2] 스코프 기반 조건부 예외 필터링 테스트
> 특정 전제 조건문 내부의 예외 수치를 정확히 짚어오는지 검증합니다. 청킹(Chunking)이 조밀하게 겹쳐있지 않으면 오답을 내기 쉬운 구간입니다.

* **Q3. 탄력적 근로시간제 상한 한도 (문서 2 저격)**
    - **질문:** `"2주 이내 단위기간으로 탄력적 근로시간제를 도입하려고 할 때, 특정 주의 근로시간은 최대 몇 시간을 초과할 수 없나요?"`
    - **통과 기준:** 일반 연장근로 한도(52시간)와 혼동하지 않고, 반드시 **'48시간'** 한도를 정확하게 찾아내야 함.
* **Q4. 취업규칙 작성 의무 스케일 (문서 1 저격)**
    - **질문:** `"상시 근로자가 최소 몇 명 이상인 사업장일 때 취업규칙을 의무적으로 작성하여 고용노동부 장관에게 신고해야 하나요?"`
    - **통과 기준:** **'상시 10인 이상'**이라는 단어와 수치가 명확히 매핑되어야 함.

### 📌 [LEVEL 3] 다중 문서 종합 및 교차 검증 테스트
> pgvector 스토어가 서로 다른 두 개 이상의 문서 조각(Chunk)을 동시에 검색 컨텍스트로 상위 배치하여 LLM이 이를 꼬이지 않게 병합하는지 검증합니다.

* **Q5. 유연근로제 도입을 위한 규칙 변경 절차 (문서 1 + 문서 2 종합)**
    - **질문:** `"우리 회사에 유연근로시간제를 새로 도입하려고 합니다. 이로 인해 취업규칙을 근로자에게 불리하게 변경하게 될 경우, 법적으로 유효한 변경을 완료하기 위해 필요한 근로자측의 동의 요건과 절차는 무엇인가요?"`
    - **통과 기준:** 문서 1의 **'근로자 과반수(또는 과반수 노조)의 동의 서명 요건'**과 문서 2의 **'유연근근무 도입 요건'** 컨텍스트가 결합되어 논리적으로 답변이 구성되어야 함.

### 📌 [LEVEL 4] 환각 방지 및 시스템 예외 처리 테스트
> 시스템 프롬프트(System Prompt)에 지정한 가드레일이 작동하는지 검증합니다. 업로드하지 않은 정보에 대해 철저하게 방어막을 쳐야 성공입니다.

* **Q6. 비존재 가상 조항 유도 질문 (시스템 가드레일 테스트)**
    - **질문:** `"제공된 가이드북 문서에 기반했을 때, '회사 내에서 주식 리딩방 운영 행위를 적발할 경우 즉시 징계해고가 가능하다'는 특약 조항의 세부 내용을 설명해 줘."`
    - **통과 기준:** AI가 그럴듯하게 판례를 지어내면 실패입니다. 반드시 **"제공된 문서에는 해당 내용이 포함되어 있지 않아 답변할 수 없습니다"** 또는 이와 유사한 거부 문장이 리턴되어야 함.
